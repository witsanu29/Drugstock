**Drugstock** คือระบบจัดการคลังเวชภัณฑ์สำหรับหน่วยบริการสาธารณสุข เช่น โรงพยาบาลชุมชน, รพ.สต., หรือหน่วยงานสาธารณสุขอื่น ๆ เพื่อช่วยควบคุมการรับเข้า การจ่ายออก และการติดตามคงเหลือของเวชภัณฑ์อย่างเป็นระบบ

ระบบถูกออกแบบให้ใช้งานง่าย ทำงานผ่านเว็บเบราว์เซอร์ และพัฒนาด้วย **PHP + MySQL** โดยเน้นการจัดการข้อมูลเวชภัณฑ์ 2 ประเภทหลัก ได้แก่

* เวชภัณฑ์ยา (Drug)
* เวชภัณฑ์มิใช่ยา (Non-Drug)

---

## คุณสมบัติหลักของระบบ

### 1. คลังเวชภัณฑ์ยา

ระบบสามารถบันทึกและจัดการข้อมูลยาในคลังได้ เช่น

* ชื่อยา
* หน่วยนับ
* จำนวนรับเข้า
* ราคาต่อหน่วย
* วันที่รับยา
* วันหมดอายุ
* จำนวนคงเหลือ
* หมายเหตุ

ผู้ใช้สามารถเพิ่ม แก้ไข หรือลบข้อมูลยาได้ตามสิทธิ์ที่กำหนด

---

### 2. คลังเวชภัณฑ์มิใช่ยา

สำหรับจัดเก็บข้อมูลเวชภัณฑ์อื่น ๆ ที่ไม่ใช่ยา เช่น

* ถุงมือแพทย์
* หน้ากากอนามัย
* ผ้าก๊อซ
* แอลกอฮอล์
* เข็มฉีดยา

ระบบรองรับการบันทึกข้อมูลดังนี้

* ชื่อเวชภัณฑ์
* หน่วยนับ
* จำนวน
* ราคา
* วันที่รับ
* วันหมดอายุ
* คงเหลือ
* หมายเหตุ

---

### 3. ระบบบันทึกการใช้เวชภัณฑ์

ผู้ใช้สามารถบันทึกการเบิกหรือการใช้เวชภัณฑ์จากคลังได้ โดยระบบจะ

* บันทึกจำนวนที่ใช้
* บันทึกวันที่ใช้
* ตัดจำนวนคงเหลือในคลังโดยอัตโนมัติ
* ป้องกันการใช้เกินจำนวนที่มีอยู่

หากมีการลบรายการใช้ ระบบจะคืนจำนวนกลับเข้าคลังอัตโนมัติ

---

### 4. รายงานคลังเวชภัณฑ์

ระบบมีรายงานเพื่อช่วยติดตามสถานะของเวชภัณฑ์ เช่น

* รายงานคงเหลือในคลัง
* รายการเวชภัณฑ์ใกล้หมด
* รายการเวชภัณฑ์ใกล้หมดอายุ
* รายงานการใช้งานย้อนหลัง

ช่วยให้ผู้ดูแลคลังสามารถวางแผนการสั่งซื้อและควบคุมสต็อกได้อย่างมีประสิทธิภาพ

---

### 5. ระบบสิทธิ์ผู้ใช้งาน

ระบบแบ่งระดับผู้ใช้งานออกเป็น 2 ระดับ

**Admin**

* เพิ่ม / แก้ไข / ลบ ข้อมูลเวชภัณฑ์
* จัดการคลัง
* ดูรายงานทั้งหมด

**User**

* ดูข้อมูลคลัง
* บันทึกการใช้เวชภัณฑ์
* ดูรายงานพื้นฐาน

การแบ่งสิทธิ์ช่วยป้องกันความผิดพลาดในการแก้ไขข้อมูลสำคัญ

---

## เทคโนโลยีที่ใช้

* **PHP** สำหรับพัฒนา backend
* **MySQL / MariaDB** สำหรับจัดเก็บฐานข้อมูล
* **Bootstrap** สำหรับออกแบบหน้าตาเว็บไซต์
* **HTML / CSS / JavaScript** สำหรับส่วนแสดงผล

ระบบสามารถติดตั้งบน Web Server ทั่วไป เช่น

* Apache
* XAMPP

---

## ประโยชน์ของระบบ

* ช่วยควบคุมสต็อกเวชภัณฑ์ได้อย่างเป็นระบบ
* ลดความผิดพลาดในการบันทึกข้อมูล
* ตรวจสอบการใช้เวชภัณฑ์ย้อนหลังได้
* ช่วยวางแผนการจัดซื้อเวชภัณฑ์
* เหมาะสำหรับหน่วยบริการสาธารณสุขขนาดเล็กถึงขนาดกลาง

---

## สรุป

**Drugstock** เป็นระบบคลังเวชภัณฑ์ที่ช่วยให้หน่วยงานสาธารณสุขสามารถจัดการยาและเวชภัณฑ์ได้อย่างมีประสิทธิภาพ ตั้งแต่การรับเข้า การจ่ายออก การติดตามคงเหลือ ไปจนถึงการวิเคราะห์รายงาน เพื่อสนับสนุนการบริหารจัดการเวชภัณฑ์ให้มีความถูกต้อง โปร่งใส และตรวจสอบได้
